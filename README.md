# Regional Sales Performance Analysis 

A data analysis project examining retail sales data to identify 
revenue trends, regional performance, and key business insights.


# Project Overview
This project analyses regional retail sales data to uncover 
top-performing regions, seasonal trends, and factors driving 
revenue growth — delivering clear, actionable business insights.


## Dataset
  Source: Superstore Sales Dataset
  Features: Region, sales, profit, product category, 
  order date, customer segment

# Key Findings
 Identified top and bottom performing regions by revenue
 Uncovered seasonal sales trends across product categories
 Highlighted profit margins by region and product type
 Delivered recommendations to improve underperforming regions
